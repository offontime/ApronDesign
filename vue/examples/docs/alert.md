# Alert 警示

## 基础使用
:::demo
```html
<Alert>这是一条消息警示</Alert>
```
:::

## 感情
:::demo
```html
<Alert type="info" content="消息警示" />
<Alert type="success" content="成功警示" />
<Alert type="warning" content="警告警示" />
<Alert type="error" content="错误警示" />
```
:::

## 宽度占满整行
:::demo
```html
<Alert fluid>这是一条消息警示</Alert>
```
:::
